### Aircraft Performance Categories

| Method        | Description                          |
| :----------   | :----------------------------------- |
| `PER A`       |     Speed 90 knots or less           |
| `PER B`       |  Between 91 and 120 knots            |
| `PER C`       |      Between 121 and 140 knots       |
| `PER D`       |      Between 141 knots and 165 knots |
| `PER E`       |      Speed 166 knots or more         |
